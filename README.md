# Frontend Mini Projects

# Welcome to Frontend Mini Projects! 

This repository contains multiple small frontend projects built using HTML, CSS, and JavaScript. Each project is designed to help practice and demonstrate different frontend concepts.

Projects

# Calculator
 A simple calculator with basic operations (add, subtract, multiply, divide).

# Tic-Tac-Toe
 Classic Tic Tac Toe game to play with a friend.

# Clock
 Real-time digital clock showing current time.

# Snake Game
 Interactive snake game using JavaScript.

(Click on each project to open its folder.)

# How to Run

# Clone this repository:
git clone https://github.com/Shakir2005-s/frontend-mini-projects.git

Open the folder of the project you want in a web browser (double-click the index.html file).
